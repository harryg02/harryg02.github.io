Github Link

Calvin Finds is a lost and found app for Calvin University students, written in React Native. My role in this development team was the UI/UX designer and one of the front-end developer.

1

Brainstorming &
Inspiration

- Clean design, inspired by modern social media apps, but posting lost/found items instead.

- Consistent design. Long rectangular shapes indicates input fields, rounded shapes indicates clickable buttons.

- Designed with accessibility and usability in mind. Use Bright yellow as primary button colors. large, bold text as primary font. Huge Buttons for easy user interaction.
