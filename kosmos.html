Kosmos Resort Website Redesign & Development

Kosmos Stargazing Resort & Spa is a resort for astrotourism located in Alamosa, CO that blends luxuary and sustainibility. The project focused on enhancing the current website and develop a custom booking widget.

Visit Website

Technology

Webflow, JavaScript, Figma, Mews API

Service

Web Design, UX design, Web Development, API integration, SEO

Result

Collaborated with marketing to roll out the redesign, contributing to an average uplift of +78 daily unique visitors (~62 % increase) and decreased 8% bounce rate.

Project Overview

Kosmos Stargazing Resort is a Colorado-based luxury destination designed for astronomy lovers that blends high-end hospitality with immersive night-sky experiences. Early community backing through a successful crowdfunding campaign helped launch construction, with the first villas opening in Spring 2025 and strong advance reservations from early supporters.
The resort’s original website didn’t match the ambition of the brand. It was visually dated, light on storytelling, and inefficient for showcasing villas, location, and availability. Key gaps included a fragmented booking flow, limited mapping/location context, and tiny default fonts that does not match to WCAG AA standard.

Objectives:

As the designer, my role encompassed:

  1. Redesign and enhance story telling of the website and aesthetics
  2. Build a custom booking widget to streamline the reservation journey that is integrated to our branding and enhancing villa selection.
  3. Enhancing the site's SEO to improve visibility and reach.
  4. Fix the small fonts and contrast issue on the original website to comply to WCAG guidelines.

Design

Homepage - Hero Section

  • Tagline for marketing
  • Big calls-to-action button with strong color contrast to draw users attention, because strong colors would pop up in user's periphery vision, guiding the user to the booking process
  • Enhancing the site's SEO to improve visibility and reach.
  • A galaxy video background to echo with the astrotourism theme

Homepage - Villa Section

  • Key visual images and features to showcase the villa
  • We chose two vertically stacked villa cards, instead of the tabbed layout, so guests can discover both options by simply scrolling, no extra clicks required.
  • Lower interaction cost: Scrolling (native, continuous) beats clicking a tab (discrete, interruptive), especially on mobile.
  • Behavior match: Modern users expect to scroll (e.g. tiktok, instagram), not hop between tabbed views.
  • "Learn more" button if the user want to see more images and details
  • "Book this villa" button continues the booking process, opens the booking widget

Booking Widget

  • Jumps to the first available date after opening the widget,S so the user doesn't have to navigate manually
  • Marks the checkout only dates, they are only selectable after the user to select the first date, eliminating confusion
  • Consistent branding and design with the rest of the page
  • Written in JavaScript
  • Integrated with Mews API to pull the dates from Mews hotel management system
